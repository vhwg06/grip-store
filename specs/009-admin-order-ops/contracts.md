# Contracts: Admin Order Operations

Module `009-admin-order-ops` mong đợi các contract nhóm sau:

## Orders

- order list with filter/search/pagination
- order detail with timeline and derived status fields
- allowed admin transitions

## Refunds

- refund queue list
- approve/reject endpoints
- backend-owned eligibility and persistence semantics

## Derived Outputs

- status labels, timeline entries, and refund state must be server-driven
