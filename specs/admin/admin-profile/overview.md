# Admin Profile Overview

## Intent

`admin-profile` là domain self-management cho current admin:

- profile identity
- password hygiene
- two-factor readiness
- backup contact methods
- recent access trust

## Actors

- `Admin / Current Operator`

## Boundaries

- `admin-profile` là self-account management
- `user` là management của account khác

## Relationships

- có thể chia sẻ identity primitives với `user`
- không sở hữu customer commerce context
